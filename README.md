# 使用方法

1. コンパイル
   
   GetHTML.cファイルをコンパイルしてください。
   
   ```例
   $gcc GetHTML.c -o ファイル名
   ```

2. 実行
   
   URLをパラメータとして入力し、生成されたファイルを実行してください。
   
   ```例
   $./ファイル名　URL
   ```

3. 実行結果
   
   無事実行できれば、ターミナルに”***Download Success***.”が表示され、ローカルにhttp.htmlファイルが保存されます。
